# BMI-Calculator-Flask
Calculate BMI of Human body and Gives the statement


# Inputs -
Height : Number (cm)

Weight : Number (Kg)

# output -

0 - Extremely Weak

1 - Weak

2 - Normal

3 - Overweight

4 - Obesity

5 - Extreme Obesity
